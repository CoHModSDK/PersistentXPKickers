# Persistent XP-Kickers

## 📜 Description

Persistent XP Kickers is a mod for `Company of Heroes Relaunch` built on top of `CoHModSDK` which
keeps the XP kickers display when the maximum command points are reached.

## 🔧 Installation

1. Install the `CoHModSDK` loader in your game directory
2. Copy `PersistentXPKickers.dll` into the `mods` folder
3. Add `PersistentXPKickers.dll` to `CoHModSDKLoader.ini`
4. Start the game
